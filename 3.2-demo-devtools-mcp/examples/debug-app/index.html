Debug Demo App

A deliberately broken app with intentional issues for debugging practice

Issue 1: Layout Thrashing

This button causes repeated DOM reads and writes in rapid succession, triggering layout recalculations for each operation. DevTools will show excessive reflows.

Issue 2: Undefined Variable Error

This button will cause a JavaScript error by trying to access a variable that doesn't exist. Check the console for the error message.

Issue 3: Slow Network Request

This button makes a request to a deliberately slow API endpoint (takes 3-5 seconds). Watch the Network tab to see the slow response time.

Issue 4: Unoptimized Images

These placeholder images are not optimized and have no explicit dimensions. This can cause layout shift and slow loading.

Issue 5: Synchronous JavaScript

This will run a synchronous loop that blocks the main thread for several seconds, freezing the UI. Check the Performance tab to see main thread blocking.
