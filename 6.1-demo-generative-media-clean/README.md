# 6.1 Generative Media for Your Apps — Clean Version

## Overview

This is the **starting point** for the Generative Media lesson. This module explores how to integrate AI-generated images, video, and audio into your applications using Google's models in AI Studio.

## What You'll Explore

Three generative media capabilities:
1. **Image Generation** (Nano Banana) — Quick image creation and editing
2. **Video Generation** (Veo) — Video with synchronized audio and dialogue
3. **Audio Generation** (Advanced TTS) — Natural-sounding speech with emotional nuances

## Prerequisites

- A Google account with access to [AI Studio](https://aistudio.google.com)
- A Gemini API key (free tier available)

## Getting Started

1. Visit [AI Studio](https://aistudio.google.com) and get an API key
2. Try the live demos linked in `6.1-demo-generative-media/README.md`
3. Experiment with prompts for each media type
4. Consider how these APIs could integrate into your own applications
