# 2.1 Frontend Design Skill — Clean Version

## Overview

This is the **starting point** for the Frontend Design Skill demo. You have a generic "before" landing page and the skill file — your goal is to use Claude with the skill to transform it into a distinctive, intentional design.

## What You Have

- `before/index.html` — A generic AI-generated landing page (the "AI slop" version)
- `.claude/skills/frontend-design.md` — The Frontend Design Skill that teaches Claude design principles

## What You'll Build

Using the Frontend Design Skill, transform the generic page into one with:
- Custom typography (serif + sans-serif pairing)
- Cohesive color system via CSS variables
- Orchestrated entrance animations
- Micro-interactions on hover states
- Responsive, accessible layout

## Getting Started

1. Open `before/index.html` in your browser to see the generic version
2. Read `.claude/skills/frontend-design.md` to understand the skill
3. Ask Claude: *"Using the frontend-design skill, transform before/index.html into a distinctive, professional landing page"*
4. Compare the result to the original
